# Detector de Pessoas

## Descrição

Este projeto utiliza o modelo YOLOv3 para detectar e contar o número de pessoas em um vídeo capturado pela webcam.

## Requisitos

- Python 3.x
- OpenCV
- NumPy

## Instalação

1. Clone o repositório.
2. Instale as dependências:
   ```bash
   pip install -r requirements.txt
